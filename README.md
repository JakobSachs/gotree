# gotree
A small exercise in Go 

# Installation
Clone the repo:
```bash
git clone https://github.com/JakobSachs/gotree
cd gotree
```
Build:
```
go build
```

and run:
```
gotree 
```

# Options

Sofar only the only option is the depth of the file system tree to be printed:
```
./gotree -level 5 ~
```
